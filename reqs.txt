Unicorn SoftInternship Training RequirementWeb Application Development (Speech-to-Text System)1. Project Title“VerbaSense”Speech-to-Text Web Application Using Whisper Model2. Project OverviewThis internship project focuses on designing and developing a modern web application that converts human speech into written text using OpenAI’s Whisper speech recognition model.The application allows users to speak through their microphone, process the audio on the backend using Node.js integrated with a Whisper wrapper, and return accurate transcribed text to the frontend in real time or near real time.Additionally, the system includes a secure authentication mechanism and MongoDB database integration to manage users and store transcription history.This project aims to provide hands-on experience in full-stack web application development, including frontend development, backend API design, AI model integration, database management, and real-time data processing.3.
